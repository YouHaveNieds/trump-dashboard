# Trump Command Center

Real-time monitoring dashboard for Trump AI assistant and agent team.

## Stack

- Vanilla HTML/CSS/JS (no build step)
- Supabase Realtime for live updates
- Lucide-style SVG icons (inline)
- Deployed on Netlify

## Local Preview

Open `index.html` in any browser. Requires internet for Supabase connection.

## Deploy

Push to main branch on GitHub. Netlify auto-deploys from the publish directory.
